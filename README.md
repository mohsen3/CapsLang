# CapsLang

## What it does?
Switch input method using Caps Lock key on Windows

## How to use it?
Either of these ways:

- Install [AutoHotKey](https://www.autohotkey.com/), then download and run `CapLang.ahk`.

- Download and run the exe file from the [realeses](https://github.com/mohsen3/CapsLang/releases) section.
